# Getting Started with the api

## Available Scripts

In the project directory, you can run:

### `pnpm run dev`

```
pnpm run dev
```

Runs the app in the development mode.


### `pnpm run build`

Start the script for build the app from typescript to javascript

```
pnpm run build
```

### `pnpm run test`

Run all tests in the app

```
pnpm run test
```
